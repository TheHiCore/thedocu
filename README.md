## Overview

**Docu** is a modern document creation tool designed to generate clean, professional documents using reusable templates and a live visual editor.

It allows users to customize document content, background images, signatures, and layout positioning through an intuitive interface, while providing a real-time preview of the final output.

Docu focuses on clarity, precision, and professional presentation.

---

## Key Features

- Template-based document generation
- Live document preview
- Background image support
- Signature upload and positioning
- Color and layout customization
- Clean, structured user interface
- Export-ready document output

---

## Technology Stack

Docu is built using the following technologies:

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui

---

## Purpose

The goal of Docu is to simplify the creation of structured and branded documents without requiring design or layout expertise, while still offering precise control over the final result.
